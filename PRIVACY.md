# Privacy Policy

Effective date: June 8, 2026

Markdown Table Editor for JetBrains IDEs runs locally inside the user's IDE.

The plugin does not collect, store, sell, or transmit personal data. It does not send Markdown files, table contents, usage data, telemetry, analytics, credentials, or IDE metadata to the plugin author or to third-party services.

The plugin uses JetBrains IntelliJ Platform APIs to read and update the currently edited document only when the user invokes plugin actions such as aligning a Markdown table, sorting rows, inserting a table, or converting selected CSV/TSV text.

If you have privacy questions, open an issue in the project repository:

https://github.com/krotname/IdeaMarkdownTableEditor/issues
